Marca: COMTAC
Modelo: COMTAC USB-A 3.0 AMAF 20 METROS
Part Number: 9376

O Cabo Extensor USB 3.0 AM AF com 3 amplificadores de sinal é a solução para conectar dispositivos de conexão USB 3.0 como câmeras, webcams, pendrives, mouses, teclados, impressoras, etc a uma longa distância sem perda de velocidade.
A velocidade de transferência de dados é de até 5.0 Gbps.

CABO EXTENSOR USB PARA WEBCAM LOGITECH
Este cabo é compatível com as Webcams da marca Logitech.

INTERFACE
USB 3.0 SuperSpeed, compatível com os padrões anteriores USB 2.0 e USB 1.1;
Velocidade máxima de transferência de dados: até 5 Gbps.

CONECTORES
USB A macho e USB A fêmea.

SISTEMAS OPERACIONAIS COMPATÍVEIS
Windows, Linux e Mac OS;
Plug e Play.

OBSERVAÇÃO: CONECTOR MICRO USB
Utilizado para dispositivos que precisam de energia adicional;
Alguns dispositivos exigem maior consumo de energia. Utilize uma fonte 5V 2A com conector Micro USB (não inclusa) para ter um melhor desempenho e funcionamento .

CARACTERÍSTICAS
Amplificadores de sinal: com o prolongamento do cabo, o sinal USB perde velocidade de transmissão.
Por isso, foi instalado 3 amplificadores de sinal na extensão do cabo que permite conectar dispositivos USB 3.0 a uma longa distância sem perda de velocidade.
