OBS: Qual é a diferença entre os modelos UC padrão e os modelos que são certificados para Microsoft Skype for Business?

Os modelos que são certificados para Microsoft Skype for Business (Lync) suportam Microsoft Skype for Business no modo pronto para usar. Esses modelos suportam recursos para atender e terminar chamadas, ajustar volume e colocar no mudo. Se você não estiver usando o Microsoft Skype for Business, use o modelo UC padrão.

Para configurar os recursos para controle de chamada remota e status de presença, instale o Jabra Direct.
