RECURSOS
VINTE BOTÕES AJUSTADOS PARA MMO
Personalização evoluída
Pré-configurado para um ótimo uso de MMO, e entregue pronto.
O G600 inclui três botões principais, doze botões para o polegar, uma roda de rolagem clicável que se inclina e um comutador G para dobrar e dominar.
Use o Logitech Gaming Software opcional para programação e personalização completas.

PAINEL PARA O POLEGAR COM BOTÕES ANATÔMICOS
Preciso e intuitivo
Dê ao polegar acesso imediato as doze botões habilidosos do MMO.
O exclusivo design de botões automáticos do painel para o polegar foi desenvolvido com base em sugestões diretas de jogadores do MMO e informações de desenvolvedores de jogo do MMO.
Os testes de campo resultaram em dois de seis botões conjuntos cuidadosamente posicionados, ajudando a reduzir o tempo de aprendizado e o tempo de ação.
E a força de atuação foi cuidadosamente ajustada para minimizar cliques incorretos.

BOTÃO G-SHIFT
Dobre o controle
Use o botão G-Shift para dobrar o número total de funções personalizáveis.
Na configuração padrão, pressionar o botão G-Shift ativa uma segunda função para cada tecla G. Ou você pode definir o G-Shift para alterar temporariamente o DPI e assim obter uma precisão extra do cursor.
