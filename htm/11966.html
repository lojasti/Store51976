Marca: MIKROTIK
Modelo: CRS326-24G-2S+RM
Part Number: CRS326-24G-2S+RM

O Cloud Router Switch crs326-24g-2s+rm é um Switch Gigabit Ethernet de 24 portas com SWOS e RouterOS com duas portas SFP+.
O Cloud Router possui dois sistemas operacionais, no entanto, com o recurso Dual Boot é possível escolher o sistema operacional de sua preferência: RouterOS ou SwOS.
Se você é uma pessoa que prefere recursos simplificados, use o SwOS, no entanto, se você prefere usar o switch com todos os recursos, incluindo os recursos de camada 3 e máxima potência, escolha o RouterOS.
Além disso, o Cloud Router Switch crs326-24g-2s+rm permite gerenciar o encaminhamento porta a porta, aplicar filtro MAC, configurar VLANs, espelhar tráfego, aplicar limitação de largura de banda e até mesmo ajustar alguns campos de cabeçalho MAC e IP.

Características:

Licença Level 5 do RouterOS
Sistema operacional : RouterOS / SwitchOS
Tamanho de RAM : 512 MB
Tamanho de armazenamento : 16 MB
Tensão de entrada do conector : DC 10-30 V
Entrada PoE : Passivo
Tensão de entrada PoE 10-30 V
Portas 10/100/1000 : 24
Portas SFP + : 2
Porta serial : RJ45.

Garantia de 12 meses para defeito de fabricação, direto com a fabricante.
