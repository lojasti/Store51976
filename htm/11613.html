Características:
Disponível no formato quadrado
As molduras foram projetadas com sistema de ampliação do ângulo de dispersão
Possuem pontos exclusivos para a fixação dos mais variados tipos de trafos, não permitindo que sejam colocados em pontos que causam danos a estrutura
Cada uma das torres é responsável pelo travamento de uma das laterais da arandela o que proporciona fixação de 100% de sua base no teto
O projeto permite flexibilidade na furação do local de instalação (se a a furação não ficar perfeita conforme o gabarito, ainda assim a arandela irá se encaixar sem que o trabalho precise ser refeito).
Permite o travamento desde a menor até a maior estrutura de teto (de 1mm a 30mm)
Terminais de engate rápido
Conjunto móvel otimizado para a reprodução de baixas frequências

Especificações Técnicas:
Formato : Quadrado
Tipo : Coaxial
Impedância : 8 Ohms
Potência Máxima : 200W
Tweeter : Neodímio e domo de seda com sistema pivotante. Proteção contra sobre carga.
Divisor : Duas vias
Crossover : Corte de frequência para o alto-falante e tweeter
Resposta de frequência : 50 - 20KHz
SPL (dB/W/m) : 89
